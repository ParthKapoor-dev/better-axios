# 📦 Installation

You can install `better-axios` via npm or yarn:

```bash
npm install @parthkapoor/better-axios
# or
yarn add @parthkapoor/better-axios
````

---

## ✅ Peer Dependencies

Make sure you also have:

* `axios` v1.x
* `typescript` (if you're using TS)

---

## 🧪 Minimal Setup

Here’s how to set up the library in your project:

```ts
import { AxiosApi } from "@parthkapoor/better-axios";

const api = new AxiosApi({
  baseURL: "https://api.example.com",
  tokenPrefix: "Bearer", // optional
});
```

Now you can use methods like `get`, `post`, `put`, `delete` with type-safe responses.
